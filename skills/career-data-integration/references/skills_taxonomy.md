# Skills Taxonomy Reference

Comprehensive skills classification system for career matching across 14 industry sectors.

## Overview

This taxonomy provides a hierarchical, sector-neutral skills framework for matching candidates to job opportunities across all industries, not just technology.

### Industry Sectors (14 Total)

2. Healthcare & Medical

7. Hospitality & Tourism

12. Marketing & Advertising
13. Energy & Utilities
14. Agriculture & Food Services

## Usage in Matching Algorithm

### Skill Similarity Scoring

```python
def calculate_skill_match(user_skill, job_skill):
    # Exact match
    if user_skill.id == job_skill.id:
        return 1.0

    # Synonym match
    if user_skill.name in job_skill.synonyms:
        return 0.95

    # Parent-child relationship
    if user_skill.parent_skill == job_skill.id:
        return 0.80
    if job_skill.parent_skill == user_skill.id:
        return 0.70

    # Related skills
    if user_skill.id in job_skill.related_skills:
        return 0.60

    # Same category
    if user_skill.category == job_skill.category:
        return 0.40

    # No match
    return 0.0
```

### Proficiency Level Matching

```python
proficiency_order = ["beginner", "intermediate", "advanced", "expert"]

def proficiency_match(user_level, required_level):
    user_index = proficiency_order.index(user_level)
    required_index = proficiency_order.index(required_level)

    if user_index >= required_index:
        return 1.0  # User meets or exceeds requirement
    else:
        gap = required_index - user_index
        return max(0.0, 1.0 - (gap * 0.25))  # Penalty for skill gap
```
